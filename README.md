## жҰӮеҝө

AIPLC жҳҜдёҖеҖӢ Web-based зҡ„ PLC й–ӢзҷјиҲҮжЁЎж“¬е№іеҸ°пјҢж ёеҝғзү№й»һпјҡ

- **C иӘһиЁҖй–Ӣзҷј** вҖ” дёҚиө°еӮізөұ Ladder/STпјҢз”Ё C еҜ« PLC зЁӢејҸпјҢе°Қ LLM еҸӢеҘҪ
- **AI Code Assistant** вҖ” дёІжҺҘ Claude APIпјҢйҖҸйҒҺ Chat д»Ӣйқўиј”еҠ©еҜ«зЁӢејҸ
- **MCP ж•ҙеҗҲ** вҖ” Agent еҸҜйҖҸйҒҺ MCP ж“ҚдҪңз·ҡи·Ҝең–пјҲж”ҫзҪ®е…ғд»¶гҖҒжҺҘз·ҡгҖҒй…ҚзҪ® I/Oпјү
- **STM32 жЁЎж“¬** вҖ” еҫҢз«Ҝз”Ё QEMU жЁЎж“¬ STM32пјҢзңҹжӯЈи·‘з·ЁиӯҜеҫҢзҡ„ ELF
- **е·ҘжҺ§е…ғд»¶еә«** вҖ” жҢүйҲ•гҖҒжҢҮзӨәзҮҲгҖҒз№јйӣ»еҷЁгҖҒйӣ»зЈҒй–ҘгҖҒйҰ¬йҒ”гҖҒVFD зӯү

```
дҪҝз”ЁиҖ…жҸҸиҝ°йңҖжұӮ вҶ’ AI з”ҹжҲҗжҺҘз·ҡең– + C зЁӢејҸ вҶ’ з·ЁиӯҜ вҶ’ QEMU жЁЎж“¬ вҶ’ еҚіжҷӮзңӢзөҗжһң
```

## PLC HAL API

дҪҝз”ЁиҖ…еҜ«зҡ„ C зЁӢејҸдҪҝз”Ё `aiplc.h` жҸҗдҫӣзҡ„ PLC йўЁж ј APIпјҡ

```c
#include "aiplc.h"

void PLC_Init() {
    Serial_Print("Ready\n");
}

void PLC_Scan() {
    bool start = DI_Read(0);    // ж•ёдҪҚијёе…Ҙ
    bool stop  = DI_Read(1);

    static bool running = false;

    if (stop)  running = false;
    if (start) running = true;

    DO_Write(0, running);       // ж•ёдҪҚијёеҮә вҶ’ жҺҘи§ёеҷЁ
    DO_Write(1, running);       // жҢҮзӨәзҮҲ
}
```
